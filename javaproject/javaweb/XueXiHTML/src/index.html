img标签是图片标签，src属性可以设置图片的路径（相对路径（.表示当前文件所在的目录，..表示当前文件所在的上一级目录，文件名表示当前文件所在的目录的文件相当于./文件名但是./可以省略），绝对路径（http://ip:port/工程名/资源路径））
border边框大小
alt属性设置当前指定路径找不到图片时用来代替显示图片

表单标签是HTML用来收集用户信息所有元素的集合然后把这些信息发送给服务器
input type="text"表示单行输入框，value="输入默认值"
input type="password"表示密码输入框
input type="radio"表示单选框，name="链接多个单选框，输入名字连接起来"，checked="checked"默认选择
input type="checkbox"表示复选框
select表示下拉列表框
option表示下拉列表项，selected="selected"默认选择
textarea表示多行文本输入框，rows属性设置可以显示几行的高度，cols属性设置每行可以显示几个字符的宽度，<textarea>输入默认值</textarea>
input type="reset"表示重置按钮,value属性可以修改按钮上的文字
input type="submit"表示提交按钮,value属性可以修改按钮上的文字
input type="button"表示按钮,value属性可以修改按钮上的文字
input type="file"表示文件上传域,value属性可以修改按钮上的文字
input type="hidden"表示隐藏域,当我们要发送某些信息而这些信息不需要用户参与就可以使用隐藏域（提交的通知发送给服务器）
